Technologies that drives the web

HTML

HTML stands for Hyper Text Markup Language. It is used to give structure to web documents. Hypertext is text containing link to other texts, however it is not just hypertext nowadays, there is hypermedia as well. Markup means to annotate or mark. Language mean it has syntax and semantics.

CSS

CSS stands for Cascading Style Sheets. It is a style sheet language used to style html or xml documents. To know what CSS can do go to csszengarden.com . The name cascading comes from the specified priority scheme to determine which style rule applies if more than one rule matches a particular element. This cascading priority scheme is predictable.

JavaScript

It provides behaviour to websites. JavaScript is the world's most popular programming language. JavaScript is easy to learn.
JavaScript, often abbreviated as JS, is a programming language that conforms to the ECMAScript specification. JavaScript is high-level, often just-in-time compiled, and multi-paradigm. It has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions.
